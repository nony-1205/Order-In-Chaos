# Order-In-Chaos

An interactive 3D website for playing around with chaos theory and math visualization.

The Attractors 
Lorenz: The classic butterfly-shaped system. 
Aizawa: A sphere-like structure that loops into itself. 
Chen: A dense, double-lobed system with fast chaotic mixing. 
Halvorsen: A cyclic, triangular shape with sharp angles. 
Shimizu-Morioka: A compact, folded system. 
Rössler: A simple spiral that loops and folds over. 
Dadras: A wide, sprawling web of intersecting paths. 
Thomas: A symmetrical, flowing maze-like system.

What you can do Move around in 3D Use the control panel to change the variables in the equations and see how it alters the shape instantly.
